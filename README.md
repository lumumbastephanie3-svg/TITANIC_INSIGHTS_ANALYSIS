# Titanic Exploratory Data Analysis (EDA)

## Project Overview

This project presents an Exploratory Data Analysis (EDA) of the Titanic dataset from Kaggle. The objective is to understand passenger demographics, identify factors influencing survival, and derive meaningful insights through statistical analysis and data visualization.

## Objectives

- Explore the structure and quality of the dataset.
- Handle missing values and detect outliers.
- Perform univariate, bivariate, and multivariate analysis.
- Investigate relationships between passenger characteristics and survival.
- Communicate findings using visualizations and statistical summaries.

## Dataset

Source: Kaggle Titanic Dataset

The dataset contains information about Titanic passengers, including:

- Age
- Sex
- Passenger Class (Pclass)
- Fare
- Embarked Location
- Survival Status
- Family Information

## Analysis Performed

### Step 1: Data Exploration
- Imported required libraries
- Loaded dataset
- Examined rows and columns
- Checked data types
- Generated summary statistics
- Checked for missing values and duplicates

### Step 2: Data Cleaning
- Handled missing values
- Confirmed null value treatment
- Identified and visualized outliers

### Step 3: Univariate Analysis
- Age distribution of passengers
- Passenger counts by embarkation point
- Fare distribution

### Step 4: Bivariate Analysis
- Relationship between Fare and Passenger Class
- Relationship between Age and Survival
- Relationship between Embarkation Point and Survival

### Step 5: Multivariate Analysis
- Combined effect of Age, Fare, and Passenger Class on Survival
- Correlation analysis
- Survival rates across embarkation points considering Passenger Class

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings

- First-class passengers had higher survival rates.
- Female passengers were more likely to survive than males.
- Younger passengers generally had better survival chances.
- Passenger class and fare strongly influenced survival outcomes.
- Survival rates varied across embarkation locations.
